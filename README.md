# SMP

<p align="center"><img src="./docs/app.1.1.0.png" alt="app" /></p>

SMP는 Windows 기반 유튜브 링크로 백그라운드 재생하는 음악재생기입니다.  

상단에 유튜브 링크를 넣고 추가를 하면 리스트에 추가되고 재생을 누르면 유튜브를 방문하지 않아도 실행됩니다.

예> **https://www.youtube.com/watch?v=UbPgeoQE8JU**

## 🔧 주요 기능

- 로컬 음원 재생
- 플레이리스트 관리
- 플레이리스트 캐싱
- 시스템 트레이 실행
- 반복 기능 (1곡, 전곡, 반복)
- 유튜브 연동 (선택 기능)
- 오디오 출력 관리
- 아이팟 볼륨 업/다운, 재생/정지
